The Agent Builder is a visual interface for creating and configuring AI agents. It provides a tabbed interface to configure all aspects of an agent.

![Agent Builder Interface](../../images/agent-builder.png)

## Overview

The Agent Builder consists of eight configuration tabs:

1. **Agent Info** - Basic agent information and runtime selection
2. **Model** - LLM provider and model configuration
3. **System Prompt** - Agent instructions and behavior
4. **Output Schema** - Structured output format
5. **MCP Server** - Model Context Protocol server connections
6. **History** - Conversation history settings
7. **Version** - Agent versioning
8. **Alias** - Agent aliases

## Creating an Agent

1. Navigate to the Agent Builder page
2. Fill in the required fields in each tab
3. Click **Create Agent** to save the configuration

Required fields are marked with an asterisk (*).
